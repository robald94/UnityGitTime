README

Menu Manager Prototype Tool is a simple tool that allows for rapid access to multiple scenes for prototyping and testing.


HOW TO USE:

- Import the MenuManager_PrototypTool Folder into the Asset Folder. 
- Drag and drop the MenuManager prefab into the scene you want as main menu. (NOTE: This must have build index 0)
- In the Menu Manager you will see 3 input fields.
	- Menu Key: The key you press to access the menu.
	- Button: For custom button, leave empty for default.
	- Scenes: Add all the scenes you want to access (NOTE: You must add the scenes to the build index)
